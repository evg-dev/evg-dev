Developer HandBook

Docker Commands

Build image

docker build . -t tagname

Shell inside container, etc...

docker exec -it image sh

Grep logs

docker logs image 2>&1 | grep "string"

Get dump.sql from container

docker exec -it image pg_dump -U username database > dump.sql

Restore from dump.sql to container

cat dump.sql | docker exec …

Docker-compose Commands

Reload image and run demon

docker-compose stop image
docker-compose rm image
docker-compose up -d image

PostgresSQL Commands

Save backup to dump.sql

pg_dump --clean -U username database  > dump.sql

Restore from dump.sql

sudo -i -u postgres psql -d database < dump.sql

psql -U postgres -d database < dump.sql

psql --username username --dbname database -f dump.sql
